# MTMCLF 檔案規格書

## 1. 基本資料

| 項目 | 內容 |
|------|------|
| 檔案名稱 | MTMCLF |
| 檔案描述 | 員工基本資料檔 (邏輯檔) |
| 檔案類型 | LF (Logical File) |
| 記錄格式 | MC0 |
| 主鍵 | MC01 (員工代號) |
| 實體檔案 | MTMCPF |

## 2. 檔案功能說明

MTMCLF 為 MTMCPF 員工基本資料檔的邏輯檔案，提供：
- 員工資料的標準檢視
- 基本查詢與瀏覽功能
- 員工代號排序檢視
- 人事管理應用程式存取介面

## 7. 索引資料

- **索引名稱**：主鍵索引

## 8. 備註

### 設計考量
1. 此檔案為 MTMCPF 的邏輯檢視檔案
2. 提供標準的員工資料存取介面
3. 主要用於人事查詢和管理功能

### 參考關聯
- **實體檔案**：MTMCPF (員工基本資料)
- **部門檔案**：PA#DPF (部門代號檔)
- **參考檔案**：RERF (欄位定義)

### 資料完整性
1. 資料內容完全依賴 MTMCPF 實體檔案
2. 索引與 MTMCPF 保持同步
3. 無獨立的資料維護功能

### 使用注意事項
1. 此為邏輯檔案，不可直接寫入資料
2. 所有異動需透過 MTMCPF 實體檔案進行
3. 主要用於人事查詢和報表程式
4. 需配合員工權限控制機制使用
